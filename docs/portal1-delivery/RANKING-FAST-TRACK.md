# Ranking Fast-Track (Legal + Safe)

This is the fastest legal workflow to improve ranking readiness and indexing.

## A) Apply the SQL pack (2 minutes)
1. Open LocalWP site `portal1`.
2. Open **Database -> Adminer**.
3. Open **SQL command**.
4. Paste file contents from `portal1-ranking-index-pack.sql`.
5. Click **Execute**.

## B) WordPress post-apply steps (5 minutes)
1. `Settings -> Permalinks -> Save Changes`.
2. Open homepage + FAQ and confirm:
   - trust/disclaimer section appears,
   - outage note appears,
   - links are visible.

## C) Rank Math minimal completion (10 minutes)
For low-score pages, ensure:
- 1 focus keyword per page (already mapped),
- 1 relevant image with alt text containing page keyword,
- keyword in H1 + one H2 + first paragraph,
- at least 2 internal links.

Ignore upsell-only notices such as Content AI.

## D) Google indexing fast-track (must do manually once domain is live)
1. Connect site to Google Search Console (Domain property).
2. Submit sitemap URL: `https://yourdomain.com/sitemap_index.xml`
3. Use URL Inspection on:
   - homepage
   - download page
   - login page
   - state guide page
4. Click **Request Indexing** for each.

## E) Final trust signals (E-E-A-T)
Add these visible blocks:
- "Ultima actualizacion" date on key pages,
- "Sitio informativo no oficial" disclaimer,
- official references to `.gob.mx`.

This improves trust and reduces legal/credibility risk.
