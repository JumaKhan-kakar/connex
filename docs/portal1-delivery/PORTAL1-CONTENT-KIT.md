1. `Inicio`

4. `Estados`

- `RFC y CURP`

## 9) Tone rules

Use this tone across the site:

- clear
- practical
- informative
- simple Spanish
- short paragraphs
- easy headings

Avoid:

- overly long menu labels
- repeated phrases
- duplicate footer text
- too many calls to action in one section
- unclear academic wording

---

## 10) Global consistency rules

When placing the content into Kadence:

- use the same heading style on all pages
- keep section spacing consistent
- keep buttons visually identical
- use the same card structure on homepage sections
- use one footer structure only
- do not mix multiple menu patterns

This content kit is designed to help the rebuild stay clean and consistent from the start.
